HTML5: Figure and Figure Caption

<figure> </figure>

HTML5 intoduced new element to contain image and its caption so that the two are associated. You can have more than one image in the tag as long as they all share the same caption

<figcaption></figcaption>

This Element has been added in HTML5 to allow authors to add caption of the images.
